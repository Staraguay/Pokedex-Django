¡En este proyecto de Django podras obtener toda la informacion de cada Pokemon a detalle!

El presente proyecto de programación fue elaborado por Sebastián Taraguay como parte de una prueba de desarrollo para la empresa Caronte. La aplicación web está elaborada con el Framewrok de Django en el lenguaje de programación Python junto con HTML y CSS, su funcionamiento se da mediante el consumo de la API de Pokemón por lo cual simula a una Pokedex, para los elementos visuales se utilizo Bootstrap 5.3 junto con JavaScrip y Jquery. La aplicación cuenta con una sección de Inicio de sesión para la cual será necesario ingresar las credenciales:
Usuario: sebastian
Contraseña: taraguay
Posteriormente en caso de éxito el usuario será redirigido a la pagina de inicio en donde se muestran los primeros 20 pokemons recuperados de la API ordenados en base a su número de identificación y al final se encuentra la paginación para poder ir consultando el resto de pokemons. Dentro de esta página también se encuentra la barra de navegación y la barra de búsqueda. Cada pokemón tiene un botón que redirige al usuario al perfil de dicho pokemón y muestra más información de este. Finalmente, dentro de la barra de navegación se encuentra el enlace para cerrar sesión una vez el usuario haya terminado su consulta en la pokedex.


Autor: Sebastian Taraguay - sebastara99@hotmail.es
